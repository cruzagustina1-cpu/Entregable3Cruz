La Cruz SRL - Control Documental

Esta aplicación web simula el área de Control Documental de la empresa de transporte La Cruz SRL. La idea es poder cargar documentos (Licencia, VTV, Seguro, etc.) para choferes o unidades.

La interacción se hace desde la página (DOM) usando formularios, eventos y guardando datos en localStorage, para que no se pierdan al recargar.
